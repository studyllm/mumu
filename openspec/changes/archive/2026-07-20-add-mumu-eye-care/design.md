# 沐目 - 设计语言与视觉规范（v0.1）

## 一、设计理念

### 核心

**温暖治愈 + 极致克制**

让用户打开软件时，感觉"被温柔对待"，而不是"被工具监控"。

### 三个"不要"

1. 不要冷色调（避免医疗感、专业距离感）
2. 不要鲜艳颜色（避免焦虑感、刺激感）
3. 不要复杂布局（避免工程感、产品距离感）

### 三个"要"

1. 要大量留白（呼吸感）
2. 要米色基调（温柔感）
3. 要圆润字体（亲和感）

## 五、动效规范

```
动画时长
─────────────────────────────────────
入场        300ms（淡入）
退场        500ms（淡出）
倒计时更新  200ms（数字平滑过渡）

缓动函数
─────────────────────────────────────
入场        cubic-bezier(0.16, 1, 0.3, 1)（ease-out，柔和）
退场        cubic-bezier(0.7, 0, 0.84, 0)（ease-in，沉稳）

动效原则
─────────────────────────────────────
✅ 静态优先（无意义的动画不要做）
✅ 缓入缓出（不要弹跳）
✅ 透明度过渡为主（不要缩放、位移）

❌ 不做呼吸动画（用户已确认）
❌ 不做弹跳动画（幼稚感）
❌ 不做粒子特效（干扰感）
```

## 六、组件规范

### 滑块

```
┌─────────────────────────────────────┐
│  提醒间隔                            │
│                                      │
│  20 分钟  ◯━━━[●]━━━━━━━━━━━━━━━ 60  │
│                                      │
│  ⓘ 推荐 20-30 分钟                   │
└─────────────────────────────────────┘

规格：
- 高度        24px（滑块轨道）
- 已选部分    主强调色（薄荷绿）
- 未选部分    浅灰
- 拖拽手柄    16px 圆形，纯白 + 1px 边框 + 阴影
- 当前数值    拖拽手柄上方小气泡
- 最小值      15 分钟
- 最大值      60 分钟
- 步长        5 分钟
- 推荐区间    20-30 分钟（提示但不强制）
```

### 单选 / 复选框

```
┌─────────────────────────────────────┐
│  ☑ 开机自启                          │
│  ☑ 锁屏后自动暂停计时                 │
│  ☐ 调试模式（开发者用）              │
└─────────────────────────────────────┘

规格：
- 尺寸        16×16px
- 选中        主强调色填充 + 白色对勾
- 未选中      1px 边框 + 浅灰背景
- 标签间距    8px
```

### 按钮

```
主按钮（极少使用）
┌─────────────┐
│   测试提醒   │  ← 主强调色背景，白色文字，圆角 8px
└─────────────┘

次按钮（基本不用）
┌─────────────┐
│   取消       │  ← 透明背景，灰色边框
└─────────────┘

超小灰字（跳过按钮）
[ 跳过 ]  ← 11px，10% 不透明度，hover 时 30%
```

## 八、托盘图标规范

```
图标设计
─────────────────────────────────────
基础形状    圆形（直径 32px，渲染时 16×16）
眼睛元素    简化的椭圆眼睛 + 圆形瞳孔
颜色        主强调色（薄荷绿）

状态
─────────────────────────────────────
工作状态    眼睛睁开（瞳孔位于中央）
休息状态    眼睛闭合（一条弧线）

动画
─────────────────────────────────────
工作 → 休息  缓慢闭眼（300ms）
休息 → 工作  缓慢睁眼（300ms）

不使用动画的场景（避免过度）
─────────────────────────────────────
眨眼动画    ❌（太频繁会分散注意力）
颜色变化    ❌（托盘图标应该稳定）
```

## 十一、文案规范

```
语气
─────────────────────────────────────
✅ 肯定、温和、关怀
   "今天眼睛会更舒服"
   "你比昨天少用了 47 分钟"
   "记得看看 6 米外的风景"

❌ 命令、批评、焦虑
   "你应该休息"
   "你已经超时了"
   "用眼过度有害健康"
```

```
称谓
─────────────────────────────────────
不用"您"（太正式）
不用"你"在压力语境（"你要多休息"）
用"你"在肯定语境（"你今天休息了 5 次"）
用陈述句，不用祈使句
```

```
术语
─────────────────────────────────────
不用专业术语
  ❌ "色温 4500K"、"蓝光过滤"、"20-20-20 法则"
  ✅ "暖一点"、"温和"、"20 分钟休息一次"

不用英文术语
  ❌ "Mode"、"Setting"、"Default"
  ✅ "模式"、"设置"、"默认"
```
