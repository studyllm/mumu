# 沐目隐私政策

**最后更新**：2026-07-20
**适用版本**：沐目 v0.1.0 及以上
**生效日期**：用户首次安装之日起

沐目（"本软件"）由沐目作者（以下简称"我们"）开发。本隐私政策说明本软件如何处理你的数据。

## 我们的承诺

**本软件不上传任何数据到任何服务器。** 你的设置、使用统计、休息记录**全部存储在你的电脑本地**，我们无从访问，也无法访问。

## 本软件收集的数据

下列数据**仅存储在你电脑本地**（`%APPDATA%\沐目\` 目录下），不通过网络传输：

| 数据类型 | 存储位置 | 用途 |
|---------|---------|------|
| 应用设置（提醒间隔、工作时段、是否自启等）| `settings.json` | 记住你的偏好 |
| 每日屏幕使用时长（秒）| `stats.db` | 在主窗口展示当日累计 |
| 每日休息次数与总时长 | `stats.db` | 让你看到护眼效果 |
| 暂停记录（开始/结束/原因）| `stats.db` | 区分自动暂停和手动暂停 |

## 本软件不收集的数据

我们**不收集**任何下列数据：

- ❌ 不收集你的姓名、邮箱、手机号等个人信息
- ❌ 不收集你的 IP 地址、设备指纹、地理位置
- ❌ 不收集你电脑上其他软件的使用情况
- ❌ 不收集屏幕截图、键盘输入、鼠标轨迹
- ❌ 不调用任何第三方分析、统计、崩溃上报服务
- ❌ 不向任何远程服务器发送任何 HTTP / TCP / UDP 请求

## 第三方组件

本软件使用的所有第三方库（Tauri、React、Rust 标准库、SQLite 等）均为开源软件，运行时**不与各自维护者的服务器通信**（除非你手动打开它们的官网查看文档）。

## Windows 系统权限

为实现核心功能，本软件请求以下 Windows 权限：

| 权限 | 用途 | 是否上传 |
|------|------|---------|
| 开机自启（HKCU\...\Run 注册表项）| 你勾选"开机自启"时写入 | ❌ 不上传 |
| 前台窗口 / 显示器枚举 | 判断屏幕是否亮、是否锁屏 | ❌ 不上传 |
| GetLastInputInfo | 判断你是否离开电脑（空闲检测）| ❌ 不上传 |

我们**不申请**网络权限、麦克风权限、摄像头权限、位置权限、文件系统全局权限。

## 数据删除

你随时可通过以下方式删除本软件保存的所有数据：

1. **通过本软件卸载**（控制面板 → 程序与功能 → 沐目 → 卸载）：卸载器会询问"是否清除所有数据"，确认后会删除整个 `%APPDATA%\沐目\` 目录，包括注册表自启项
2. **手动删除**：直接删除 `%APPDATA%\沐目\` 文件夹
3. **重置设置**：设置窗口最底部"恢复默认设置"按钮会清空 `settings.json`，但保留历史统计数据

**卸载是不可逆操作。** 一旦卸载，统计数据无法恢复（因为我们没有你的备份）。

## 数据所有权

你的数据**始终归你所有**。我们没有你的数据副本，没有备份，没有日志，没有 CDN 边缘缓存。卸载即真正消失。

## 儿童隐私

本软件不针对 13 岁以下儿童设计，不收集任何年龄相关信息。如果你是未成年人，请在监护人同意下使用。
