# 沐目使用说明

> 干眼患者的护眼小工具 — 装上就忘、想起来才用。

## 安装

1. 下载安装包 `沐目_0.1.0_x64-setup.exe`，双击安装
2. 完成后会自动启动，**托盘出现绿色眼睛图标** 👁 — 安装成功
3. 卸载：控制面板 → 程序与功能 → 沐目 → 卸载。会提示清除所有设置与数据

## 主界面

- **双击托盘眼睛** 👁：打开 480×360 主窗口，显示「今日屏幕使用时长」+「眼睛休息了几次」
- 右上角 **×**：只隐藏窗口，进程不退，托盘图标仍在

## 托盘菜单（右键）

| 菜单项 | 作用 |
|--------|------|
| 打开主界面 / 显示当日数据 | 打开主窗口看当日数据 |
| 暂停 30 分钟 / 1 小时 | 暂停 X 分钟内不弹窗、不累计 |
| 暂停到明早 9:00 | 适合下班前用 |
| 设置 | 打开 800×600 设置窗口 |
| 退出 | 彻底关掉进程 |

## 强提醒长什么样

工作时段内（默认 09:00–18:00）每 20 分钟右下角弹一个 320×200 半透明小窗：标题「休息一下」+ 巨号倒计时 + 副信息「看向 6 米外的物体」。**归零自动消失** + 一声木鱼响（设置可关）。**点"跳过"**立即消失，本次不算休息。**锁屏**立即暂停；解锁后若还在工作时段会续弹。

## 设置窗口（托盘 → 设置）

- **提醒设置**：工作时间（开始/结束）/ 提醒间隔（5–60 分钟）/ 休息时长（10–60 秒）/ 完成时播放声音
- **护眼保养**：眼药水提醒（启用 + 间隔）/ 热敷提醒（启用 + 每天几点）
- **通用**：托盘快速暂停（30 分钟 / 1 小时 / 到明早 9:00）
- **高级**：开机自启

最底下还有 **「测试提醒」** 按钮 — 立刻弹一次 5 秒迷你强提醒，方便调设置时即时看效果。所有改动**即时保存**。

## 常见问题

**Q1：怎么暂停提醒？**
右键托盘眼睛 → 选「暂停 30 分钟 / 1 小时 / 到明早 9:00」。要关掉特定提醒，进设置窗口关掉「眼药水提醒」/「热敷提醒」即可。

**Q2：怎么修改工作时段？**
托盘 → 设置 → 提醒设置 → 工作时间。**结束时间必须晚于开始时间**，否则会被拒绝保存。

**Q3：怎么彻底退出？**
右键托盘眼睛 → 「退出」。会杀掉进程，下次不会自启（除非你勾了「高级 → 开机自启」）。

## 数据保存在哪里

全部存在本地：`%APPDATA%\沐目\`（`settings.json` + `stats.db`）。**不上传任何数据**，无需联网。
